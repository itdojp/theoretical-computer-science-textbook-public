# 理論計算機科学教科書 - 数学的記法統一ガイド

## 1. 基本記法

### 集合と要素

| 記法 | 意味 | 使用例 |
|------|------|--------|
| ∈ | 要素関係 | x ∈ A |
| ∉ | 非要素関係 | x ∉ A |
| ⊆ | 部分集合 | A ⊆ B |
| ⊊ | 真部分集合 | A ⊊ B |
| ∪ | 和集合 | A ∪ B |
| ∩ | 積集合 | A ∩ B |
| \ | 差集合 | A \ B |
| × | 直積 | A × B |
| P(A), 2^A | 冪集合 | P({0,1}) = {∅, {0}, {1}, {0,1}} |
| |A| | 集合の濃度 | |{1,2,3}| = 3 |
| ∅ | 空集合 | A ∩ B = ∅ |

### 論理記号

| 記法 | 意味 | 使用例 |
|------|------|--------|
| ∧ | 論理積（かつ） | p ∧ q |
| ∨ | 論理和（または） | p ∨ q |
| ¬ | 否定（でない） | ¬p |
| → | 含意（ならば） | p → q |
| ↔, ⟺ | 同値（同値である） | p ↔ q |
| ∀ | 全称量化子 | ∀x ∈ A, P(x) |
| ∃ | 存在量化子 | ∃x ∈ A, P(x) |
| ⊢ | 証明可能 | Γ ⊢ φ |
| ⊨ | 意味論的帰結 | Γ ⊨ φ |

### 数と関数

| 記法 | 意味 | 使用例 |
|------|------|--------|
| ℕ | 自然数（1から） | n ∈ ℕ |
| ℕ₀ | 非負整数 | n ∈ ℕ₀ |
| ℤ | 整数 | z ∈ ℤ |
| ℚ | 有理数 | q ∈ ℚ |
| ℝ | 実数 | r ∈ ℝ |
| ℂ | 複素数 | c ∈ ℂ |
| f: A → B | 関数の型 | f: ℕ → ℕ |
| f ∘ g | 関数の合成 | (f ∘ g)(x) = f(g(x)) |
| ⌊x⌋ | 床関数 | ⌊3.7⌋ = 3 |
| ⌈x⌉ | 天井関数 | ⌈3.2⌉ = 4 |
| log, ln | 対数（底2、自然対数） | log n, ln x |

## 2. 計算理論の記法

### チューリング機械

| 記法 | 意味 | 規約 |
|------|------|------|
| M | チューリング機械 | 大文字で表記 |
| Q | 状態集合 | 有限集合 |
| Σ | 入力アルファベット | ␣ を含まない |
| Γ | テープアルファベット | Σ ⊆ Γ, ␣ ∈ Γ |
| δ | 遷移関数 | δ: Q × Γ → Q × Γ × {L,R} |
| q₀ | 初期状態 | q₀ ∈ Q |
| qₐ, qᵣ | 受理・拒否状態 | qaccept, qreject とも表記 |
| ⟨M⟩ | Mの符号化 | 山括弧で囲む |
| ⊢ | 1ステップ遷移 | C₁ ⊢ C₂ |
| ⊢* | 遷移の反射推移閉包 | C₁ ⊢* C₂ |

### 言語とオートマトン

| 記法 | 意味 | 使用例 |
|------|------|--------|
| L | 言語 | L ⊆ Σ* |
| Σ* | クリーネ閉包 | すべての文字列の集合 |
| ε | 空文字列 | |ε| = 0 |
| w, x, y | 文字列変数 | w ∈ Σ* |
| |w| | 文字列の長さ | |abc| = 3 |
| wᴿ | 文字列の逆順 | (abc)ᴿ = cba |
| L₁L₂ | 言語の連結 | {xy | x ∈ L₁, y ∈ L₂} |
| L* | 言語のクリーネ閉包 | L* = ⋃ᵢ₌₀^∞ Lⁱ |
| DFA, NFA | 有限オートマトン | 決定性・非決定性 |
| CFG | 文脈自由文法 | Context-Free Grammar |
| PDA | プッシュダウンオートマトン | Pushdown Automaton |

## 3. 計算複雑性の記法

### 漸近記法

| 記法 | 定義 | 直観的意味 |
|------|------|------------|
| O(f(n)) | ∃c,n₀, ∀n≥n₀, g(n) ≤ c·f(n) | 上界 |
| Ω(f(n)) | ∃c,n₀, ∀n≥n₀, g(n) ≥ c·f(n) | 下界 |
| Θ(f(n)) | g(n) = O(f(n)) ∧ g(n) = Ω(f(n)) | 上下界一致 |
| o(f(n)) | lim_{n→∞} g(n)/f(n) = 0 | 真に小さい |
| ω(f(n)) | lim_{n→∞} g(n)/f(n) = ∞ | 真に大きい |

### 複雑性クラス

| 記法 | 意味 | 定義 |
|------|------|------|
| TIME(f(n)) | 時間複雑性クラス | f(n)時間で決定可能 |
| SPACE(f(n)) | 空間複雑性クラス | f(n)空間で決定可能 |
| NTIME(f(n)) | 非決定性時間 | f(n)時間で非決定的認識 |
| P | 多項式時間 | ⋃ₖ TIME(nᵏ) |
| NP | 非決定性多項式時間 | ⋃ₖ NTIME(nᵏ) |
| PSPACE | 多項式空間 | ⋃ₖ SPACE(nᵏ) |
| EXPTIME | 指数時間 | ⋃ₖ TIME(2^{nᵏ}) |
| L | 対数空間 | SPACE(log n) |
| NL | 非決定性対数空間 | NSPACE(log n) |

### 還元と完全性

| 記法 | 意味 | 使用例 |
|------|------|--------|
| A ≤ₚ B | 多項式時間還元 | SAT ≤ₚ 3-SAT |
| A ≤ₘ B | 多対一還元 | 一般的な還元 |
| A ≤ₜ B | Turing還元 | オラクル使用可 |
| NP-complete | NP完全 | L ∈ NP ∧ ∀A∈NP, A≤ₚL |
| NP-hard | NP困難 | ∀A∈NP, A≤ₚL |

## 6. 統一規則

### 変数の使い方

1. **集合**: 大文字（A, B, S, T）
2. **要素**: 小文字（x, y, a, b）
3. **自然数**: i, j, k, m, n
4. **文字列**: u, v, w, x, y, z
5. **関数**: f, g, h（小文字）
6. **定数**: c, d（小文字）
7. **確率**: p, q（小文字）

### インデックス

- 下付き文字：x₁, x₂, ..., xₙ
- 上付き文字（冪乗）：2ⁿ, xᵏ
- プライム記号：x', x''（導関数や変形を表す）

### 証明記法

- 証明開始：「証明：」または「Proof:」
- 証明終了：□ または QED
- 定義：「定義 n.m」（n章m番目）
- 定理：「定理 n.m」
- 補題：「補題 n.m」
- 系：「系 n.m」

### 数式の表示

1. **インライン数式**: 文中では簡潔に
   - 例：「時間計算量はO(n log n)である」

2. **ディスプレイ数式**: 重要な式は独立行で
   ```
   T(n) = 2T(n/2) + O(n)
   ```

3. **複数行の式**: 整列して表示
   ```
   H(X,Y) = H(X) + H(Y|X)
          = H(Y) + H(X|Y)
   ```

## 7. 注意事項

### 避けるべき記法の曖昧性

1. **集合の要素と数値の区別**
   - 悪い例：n（集合なのか数なのか不明）
   - 良い例：|S| = n（Sは集合、nはその要素数）

2. **関数と値の区別**
   - 悪い例：f = 5
   - 良い例：f(x) = 5 または f: A → {5}

3. **包含関係の明確化**
   - 使い分け：⊆（部分集合）、⊊（真部分集合）
   - 等号との組み合わせを避ける

### 記法の導入規則

1. 初出時は必ず定義を明記
2. 標準的でない記法は避ける
3. 章をまたぐ場合は再定義も検討
4. 索引に主要記法を収録

この統一ガイドにより、教科書全体で一貫した数学的記法を維持し、読者の理解を促進します。
