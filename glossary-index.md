# 理論計算機科学教科書 - 用語集・索引

## A

### Algorithm（アルゴリズム）
**定義**: 問題を解くための明確で有限な手順の集合
**英語**: Algorithm
**関連概念**: 計算複雑性、正当性、効率性
**登場章**: 第1章、第6章
**関連用語**: 計算量、時間複雑性、空間複雑性

### Asymptotic Notation（漸近記法）
**定義**: 関数の増加率を表すための記法（O, Ω, Θ記法）
**英語**: Asymptotic Notation
**詳細**: 
- **O記法**: 上界を表す f(n) = O(g(n))
- **Ω記法**: 下界を表す f(n) = Ω(g(n))  
- **Θ記法**: 上界と下界が一致 f(n) = Θ(g(n))
**登場章**: 第1章、第5章
**関連用語**: 計算複雑性、時間計算量

### Automaton（オートマトン）
**定義**: 計算を抽象化した数学的モデル
**英語**: Automaton（複数形: Automata）
**種類**:
- **DFA**: 決定性有限オートマトン
- **NFA**: 非決定性有限オートマトン
- **PDA**: プッシュダウンオートマトン
**登場章**: 第3章
**関連用語**: 形式言語、状態遷移

## B

### Big-O Notation（ビッグオー記法）
**定義**: 関数の上界を表すための漸近記法
**英語**: Big-O Notation
**数学的定義**: f(n) = O(g(n)) ⟺ ∃c > 0, ∃n₀, ∀n ≥ n₀, f(n) ≤ c·g(n)
**登場章**: 第1章、第5章
**関連用語**: 時間計算量、漸近記法

### Binary Search Tree（二分探索木）
**定義**: 各ノードが二分探索性質を満たす二分木
**英語**: Binary Search Tree (BST)
**性質**: 左の子 < 親 < 右の子
**操作**: 挿入O(log n)、削除O(log n)、探索O(log n) (平均)
**登場章**: 第7章
**関連用語**: データ構造、木構造

### BFS（幅優先探索）
**定義**: グラフの各レベルを順次探索するアルゴリズム
**英語**: Breadth-First Search
**データ構造**: キューを使用
**時間計算量**: O(V + E)
**登場章**: 第8章
**関連用語**: グラフ探索、DFS

## C

### Chomsky Hierarchy（チョムスキー階層）
**定義**: 形式言語を生成力によって分類した階層
**英語**: Chomsky Hierarchy
**階層**:
- **Type 0**: 再帰的可算言語（制限なし文法）
- **Type 1**: 文脈依存言語（文脈依存文法）
- **Type 2**: 文脈自由言語（文脈自由文法）
- **Type 3**: 正規言語（正規文法）
**登場章**: 第3章
**関連用語**: 形式言語、文法

### Church-Turing Thesis（チャーチ・チューリングの提唱）
**定義**: 直観的に計算可能な関数とチューリング機械で計算可能な関数は一致するという仮説
**英語**: Church-Turing Thesis
**意義**: 計算可能性の本質を表現
**登場章**: 第2章、第4章
**関連用語**: 計算可能性、チューリング機械

### Complexity Class（計算複雑性クラス）
**定義**: 同程度の計算資源で解ける問題の集合
**英語**: Complexity Class
**主要クラス**:
- **P**: 多項式時間で解ける問題
- **NP**: 非決定性多項式時間で解ける問題
- **PSPACE**: 多項式空間で解ける問題
**登場章**: 第5章
**関連用語**: P vs NP、計算複雑性

### Context-Free Grammar（文脈自由文法）
**定義**: 生成規則の左辺が単一の非終端記号からなる文法
**英語**: Context-Free Grammar (CFG)
**形式**: A → α （A: 非終端記号、α: 記号列）
**登場章**: 第3章
**関連用語**: 文脈自由言語、導出木

### Cook-Levin Theorem（クック・レヴィンの定理）
**定義**: SAT問題がNP完全であることを示した定理
**英語**: Cook-Levin Theorem
**意義**: NP完全性理論の出発点
**登場章**: 第5章
**関連用語**: NP完全、SAT

## D

### Decidability（決定可能性）
**定義**: 問題に対して必ず停止するアルゴリズムが存在するかどうか
**英語**: Decidability
**分類**:
- **決定可能**: アルゴリズムが存在
- **決定不能**: アルゴリズムが存在しない
**登場章**: 第2章、第4章
**関連用語**: 停止問題、計算可能性

### DFS（深さ優先探索）
**定義**: グラフを深さ方向に探索するアルゴリズム
**英語**: Depth-First Search
**データ構造**: スタック（または再帰）を使用
**時間計算量**: O(V + E)
**登場章**: 第8章
**関連用語**: グラフ探索、BFS

### Dijkstra's Algorithm（ダイクストラ法）
**定義**: 非負重み付きグラフの単一始点最短路問題を解くアルゴリズム
**英語**: Dijkstra's Algorithm
**時間計算量**: O((V + E) log V) （優先度キュー使用）
**条件**: 負の重みを持つ辺がない
**登場章**: 第8章
**関連用語**: 最短路、グリーディアルゴリズム

### Dynamic Programming（動的計画法）
**定義**: 部分問題の解を保存して効率化を図るアルゴリズム設計手法
**英語**: Dynamic Programming
**原理**: 最適部分構造、部分問題の重複
**例**: フィボナッチ数列、最長共通部分列
**登場章**: 第6章
**関連用語**: メモ化、最適化問題

## E

### Entropy（エントロピー）
**定義**: 情報源の不確実性を定量化する尺度
**英語**: Entropy
**数学的定義**: H(X) = -∑ p(x) log₂ p(x)
**単位**: ビット（対数の底が2の場合）
**登場章**: 第10章
**関連用語**: 情報量、情報理論

### Euler's Theorem（オイラーの定理）
**定義**: gcd(a,n) = 1 ならば a^φ(n) ≡ 1 (mod n)
**英語**: Euler's Theorem
**応用**: RSA暗号の数学的基礎
**登場章**: 第11章
**関連用語**: RSA、オイラー関数

## F

### Finite Automaton（有限オートマトン）
**定義**: 有限個の状態を持つ計算モデル
**英語**: Finite Automaton
**種類**: DFA（決定性）、NFA（非決定性）
**認識言語**: 正規言語
**登場章**: 第3章
**関連用語**: 状態遷移、正規言語

### Formal Language（形式言語）
**定義**: 形式的に定義された文字列の集合
**英語**: Formal Language
**構成要素**: アルファベット、文字列、言語
**登場章**: 第3章
**関連用語**: 文法、オートマトン

## G

### Graph Theory（グラフ理論）
**定義**: 頂点と辺からなるグラフ構造を研究する数学分野
**英語**: Graph Theory
**基本概念**: 頂点、辺、経路、連結性
**登場章**: 第8章
**関連用語**: ネットワーク、アルゴリズム

### Greedy Algorithm（グリーディアルゴリズム）
**定義**: 各段階で局所的に最適な選択を行うアルゴリズム
**英語**: Greedy Algorithm
**性質**: 必ずしも大域最適解を得られない
**例**: ダイクストラ法、ハフマン符号
**登場章**: 第6章
**関連用語**: 最適化、近似アルゴリズム

## H

### Halting Problem（停止問題）
**定義**: チューリング機械が与えられた入力で停止するかを判定する問題
**英語**: Halting Problem
**性質**: 決定不能
**証明**: 対角化論法
**登場章**: 第2章、第4章
**関連用語**: 決定不能性、チューリング機械

### Hash Function（ハッシュ関数）
**定義**: 任意長の入力を固定長の出力にマップする関数
**英語**: Hash Function
**性質**: 一方向性、衝突困難性
**応用**: 暗号、データ構造
**登場章**: 第7章、第11章
**関連用語**: 衝突、ハッシュ表

### Huffman Coding（ハフマン符号）
**定義**: 文字の出現頻度に基づく可変長符号化方式
**英語**: Huffman Coding
**性質**: 最適な瞬時符号
**構成**: 二分木による貪欲構成
**登場章**: 第10章
**関連用語**: 情報源符号化、符号理論

## I

### Information Theory（情報理論）
**定義**: 情報の定量化と伝達に関する数学理論
**英語**: Information Theory
**創始者**: クロード・シャノン (1948)
**基本概念**: エントロピー、相互情報量
**登場章**: 第10章
**関連用語**: 通信、符号化

## L

### Linear Programming（線形計画法）
**定義**: 線形制約下での線形目的関数の最適化問題
**英語**: Linear Programming
**解法**: シンプレックス法、内点法
**時間計算量**: 多項式時間（楕円体法）
**登場章**: 第6章
**関連用語**: 最適化、多項式時間

### Logic（論理学）
**定義**: 正しい推論の原理を研究する学問
**英語**: Logic
**種類**: 命題論理、述語論理、時相論理
**応用**: プログラム検証、AI
**登場章**: 第9章
**関連用語**: 証明、形式的手法

## M

### Master Theorem（マスター定理）
**定義**: 分割統治アルゴリズムの時間計算量を求める定理
**英語**: Master Theorem
**形式**: T(n) = aT(n/b) + f(n)
**ケース**: 3つのケースに分類
**登場章**: 第1章、第6章
**関連用語**: 分割統治、漸化式

### Model Checking（モデル検査）
**定義**: システムが仕様を満たすかを自動的に検証する手法
**英語**: Model Checking
**対象**: 有限状態システム
**論理**: CTL、LTL
**登場章**: 第9章
**関連用語**: 形式検証、時相論理

## N

### NP-Complete（NP完全）
**定義**: NPに属し、すべてのNP問題が多項式時間で還元可能な問題
**英語**: NP-Complete
**例**: SAT、クリーク、ハミルトン路
**意義**: P vs NP問題の核心
**登場章**: 第5章
**関連用語**: NP、多項式時間還元

### Non-deterministic（非決定性）
**定義**: 複数の計算パスを同時に探索する計算モデル
**英語**: Non-deterministic
**特徴**: 推測と検証
**例**: NFA、非決定性チューリング機械
**登場章**: 第2章、第3章、第5章
**関連用語**: 決定性、計算複雑性

## O

### Oracle Machine（オラクル機械）
**定義**: 外部の情報源（オラクル）にアクセス可能な計算機械
**英語**: Oracle Machine
**用途**: 相対化、計算複雑性の階層
**記法**: M^A（機械MがオラクルAを使用）
**登場章**: 第5章
**関連用語**: 相対化、計算複雑性

## P

### P vs NP Problem（P対NP問題）
**定義**: PとNPが等しいかどうかを問う計算理論の重要問題
**英語**: P vs NP Problem
**現状**: 未解決（ミレニアム問題の一つ）
**意義**: 計算の本質的限界を問う
**登場章**: 第5章
**関連用語**: 計算複雑性、NP完全

### Pumping Lemma（ポンピング補題）
**定義**: 正規言語や文脈自由言語の構造的性質を表す定理
**英語**: Pumping Lemma
**用途**: 言語が特定のクラスに属さないことの証明
**種類**: 正規言語用、文脈自由言語用
**登場章**: 第3章
**関連用語**: 正規言語、文脈自由言語

### Pushdown Automaton（プッシュダウンオートマトン）
**定義**: スタックを持つ有限オートマトン
**英語**: Pushdown Automaton (PDA)
**認識言語**: 文脈自由言語
**構成要素**: 状態、入力テープ、スタック
**登場章**: 第3章
**関連用語**: 文脈自由言語、スタック

## Q

### Quantum Computing（量子計算）
**定義**: 量子力学的現象を利用した計算パラダイム
**英語**: Quantum Computing
**特徴**: 重ね合わせ、もつれ
**複雑性クラス**: BQP
**登場章**: 第5章（言及）
**関連用語**: 量子アルゴリズム、量子暗号

## R

### Regular Expression（正規表現）
**定義**: 正規言語を記述するための形式的記法
**英語**: Regular Expression
**基本操作**: 和集合、連結、クリーネ閉包
**等価性**: DFA、NFAと等価
**登場章**: 第3章
**関連用語**: 正規言語、有限オートマトン

### RSA Cryptosystem（RSA暗号）
**定義**: 公開鍵暗号の代表的なシステム
**英語**: RSA Cryptosystem
**基礎**: 素因数分解の困難性
**発明者**: Rivest、Shamir、Adleman (1977)
**登場章**: 第11章
**関連用語**: 公開鍵暗号、素因数分解

## S

### SAT Problem（充足可能性問題）
**定義**: 論理式を真にする変数割当が存在するかを判定する問題
**英語**: Boolean Satisfiability Problem
**重要性**: 最初のNP完全問題
**変種**: 3-SAT、2-SAT
**登場章**: 第5章
**関連用語**: NP完全、論理式

### Shannon's Theorem（シャノンの定理）
**定義**: 情報源符号化定理、通信路符号化定理
**英語**: Shannon's Theorem
**内容**: 符号化の理論的限界
**意義**: 情報理論の基礎
**登場章**: 第10章
**関連用語**: エントロピー、通信路容量

## T

### Turing Machine（チューリング機械）
**定義**: 計算の数学的モデル
**英語**: Turing Machine
**構成**: 状態、テープ、ヘッド、遷移関数
**意義**: 計算可能性の標準的モデル
**登場章**: 第2章、第4章
**関連用語**: 計算可能性、チャーチ・チューリングの提唱

### Time Complexity（時間計算量）
**定義**: アルゴリズムの実行時間を入力サイズの関数として表現
**英語**: Time Complexity
**記法**: 漸近記法（O、Ω、Θ）
**重要性**: 効率性の指標
**登場章**: 第1章、第5章、第6章
**関連用語**: 空間計算量、計算複雑性

## U

### Undecidability（決定不能性）
**定義**: 決定アルゴリズムが存在しない問題の性質
**英語**: Undecidability
**例**: 停止問題、同値問題
**証明法**: 対角化、還元
**登場章**: 第2章、第4章
**関連用語**: 決定可能性、計算可能性

### Universal Turing Machine（万能チューリング機械）
**定義**: 任意のチューリング機械をシミュレートできる機械
**英語**: Universal Turing Machine
**意義**: プログラム内蔵方式の理論的基礎
**入力**: 機械の記述と入力データ
**登場章**: 第2章
**関連用語**: シミュレーション、計算可能性

## Z

### Zero-Knowledge Proof（ゼロ知識証明）
**定義**: 秘密情報を明かすことなく、その知識を証明する手法
**英語**: Zero-Knowledge Proof
**性質**: 完全性、健全性、ゼロ知識性
**応用**: 暗号プロトコル、ブロックチェーン
**登場章**: 第11章
**関連用語**: 対話型証明、暗号理論

## 記号・数学記法索引

### アルファベット・基本記法
- **Σ**: アルファベット
- **Σ***: Σ上のすべての文字列の集合
- **ε**: 空文字列
- **|w|**: 文字列wの長さ
- **L**: 言語（文字列の集合）

### 集合・論理記法
- **∅**: 空集合
- **∪**: 和集合
- **∩**: 積集合
- **\**: 差集合
- **⊆**: 部分集合
- **∈**: 要素関係
- **∀**: 全称量化子
- **∃**: 存在量化子
- **¬**: 否定
- **∧**: 論理積
- **∨**: 論理和
- **→**: 含意
- **↔**: 同値

### 計算複雑性記法
- **O(f(n))**: ビッグオー記法（上界）
- **Ω(f(n))**: ビッグオメガ記法（下界）
- **Θ(f(n))**: ビッグシータ記法（上下界一致）
- **P**: 多項式時間クラス
- **NP**: 非決定性多項式時間クラス
- **PSPACE**: 多項式空間クラス

### オートマトン記法
- **δ**: 遷移関数
- **q₀**: 初期状態
- **F**: 受理状態集合
- **⊢**: 一歩遷移関係
- **⊢***: 遷移の反射推移閉包

### 情報理論記法
- **H(X)**: エントロピー
- **I(X;Y)**: 相互情報量
- **D(P||Q)**: KLダイバージェンス
- **C**: 通信路容量

### 暗号理論記法
- **E_k(m)**: 鍵kによる暗号化
- **D_k(c)**: 鍵kによる復号
- **gcd(a,b)**: 最大公約数
- **φ(n)**: オイラー関数
- **a ≡ b (mod n)**: 合同式
